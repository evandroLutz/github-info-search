
# Github Info Search

A ReactJs tool for search information about Github user's repositories using the Github Public API.


## Author

- [@evandroLutz](https://www.github.com/evandroLutz)


## GithubPages

https://evandrolutz.github.io/github-info-search/index.html


## Deploy

To deploy this project run

```bash
  npm run deploy
```


## Install


```bash
  cd github-info-search
  npm install
```
    
## Local Run

Clone the project

```bash
  git clone https://github.com/evandroLutz/github-info-search.git
```

Enter on directory

```bash
  cd github-info-search
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
